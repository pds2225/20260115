# ExportHub MVP

글로벌 수출 인텔리전스 플랫폼

## Features

- **유망국가 추천**: AI 기반 맞춤 수출 유망국가 분석
- **매출 시뮬레이터**: 시장 진입 시나리오별 예상 매출 시뮬레이션
- **바이어 매칭**: FitScore 기반 최적 바이어 매칭
- **번역 채팅**: 실시간 자동 번역 지원 커뮤니케이션
- **계약서 생성**: 템플릿 기반 수출 계약서 자동 생성

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Recharts

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
